> 臣谨按<sup>1</sup>：《周礼》者，周公致太平之书也。孔子曰：“周监于二代<sup>2</sup>，郁郁乎文哉<sup>3</sup>！吾从周。”（《论语·八佾》）今有学生新疆入<sup>4</sup>，欲毕业桐高而不违周礼，此事大矣，不可不察！

3. **郁郁乎文哉**：文采丰盛、礼制完备的样子。
4. **新疆入**：同学外号，详见[《桐高幽默事件》](/post/2026/01/HumorousIncidents.html)。

## 一、周礼，非周罡令<sup>1</sup>之礼，乃周公之礼也

周礼，乃周公著礼制书籍。绝非“周”李浩之礼，也绝非“周”罡令之礼。今借此为据，以周礼五纲（吉、凶、军、宾、嘉）为准绳，定新疆入能否在桐高顺利毕业而不违礼。

::: note

1. **周罡令**：详见[《周罡令事件》](/post/2025/11/ZhouGanglingIncident.html)。此处与新疆入本名周李浩之"周"并列，双关"周"字以反衬周公之礼为正统。

:::

五礼者，周室纲纪之总目也。昔周公制礼作乐，定五礼以范天下：

一曰吉礼，事神祀祖之典也。郊祀天地<sup>1</sup>、宗庙享先<sup>2</sup>、祭社稷山川<sup>3</sup>，凡祈福禳灾<sup>4</sup>者属焉。五礼之尊，莫大于此。

二曰凶礼，哀死恤患之仪也。丧葬奔吊<sup>5</sup>、赈荒恤灾<sup>6</sup>，皆所以厚人伦、助教化也。死亡凶祸之事隶焉。

三曰军礼，师旅征伐之节也。出征曰“类”<sup>7</sup>，誓师曰“诰”<sup>8</sup>，大阅<sup>9</sup>、大狩<sup>10</sup>、劳军、振旅<sup>11</sup>，凡用武之事悉隶焉。所以明上下之分、严军旅之制也。

四曰宾礼，邦交朝聘之文也。诸侯朝觐天子曰“觐”<sup>12</sup>，天子接见诸侯曰“会”<sup>13</sup>，遣使往来曰“聘”<sup>14</sup>。尊卑有序、亲疏有别，邦国以宁。

五曰嘉礼，亲睦万民之仪也。冠婚之礼以成人伦，乡饮<sup>15</sup>之礼以序长幼，宴飨<sup>16</sup>之礼以洽上下。凡庆贺、颁赏、立储<sup>17</sup>、养老之属归之。

五礼行，则尊卑定、亲疏别、贵贱明、长幼序，天下可运于掌矣<sup>18</sup>。

::: note

1. **郊祀天地**：天子在南郊祭天、北郊祭地之礼。
2. **宗庙享先**：在宗庙中供奉祖先，行祭祀之礼。
3. **社稷山川**：社为土神，稷为谷神，山川指五岳四渎之神。
4. **禳灾**：祈祷以消除灾祸。
5. **奔吊**：闻丧奔赴、吊唁死者。
6. **赈荒恤灾**：发放钱粮救济灾荒、抚恤受灾民众。
7. **类**：古代天子出征前举行的祭天之礼。
8. **诰**：告诫、训勉将士的誓师之辞。
9. **大阅**：天子定期检阅军队的大典。
10. **大狩**：借狩猎之名行军事训练之实的围猎活动。
11. **振旅**：战事完毕，整顿军队班师回朝之礼。
12. **觐**（jìn）：诸侯秋季朝见天子之礼。四时朝见各有专名：春曰“朝”，夏曰“宗”，秋曰“觐”，冬曰“遇”。
13. **会**：天子因征讨等大事临时召集诸侯的会见。
14. **聘**：诸侯遣使往天子或他国致意问候之礼。使曰“聘”，大夫曰“问”。
15. **乡饮**：古代基层行政组织“乡”定期举行的饮酒礼仪，以序长幼、别尊卑、明教化。
16. **宴飨**：设宴款待宾客的大礼，飨礼重于宴礼。
17. **立储**：确立太子（储君）之礼，关乎国本。
18. **天下可运于掌矣**：天下可以在手掌中运转，喻极为容易。语出[《孟子·梁惠王上》](https://ctext.org/mengzi/liang-hui-wang-i/zhs)。

:::

## 二、新疆入行录

### 傻逼

张文昊<sup>2</sup>拒玩鸣潮，新疆入脱口而出“傻——逼”。此正犯孟子所言：“爱人者，人恒爱之；敬人者，人恒敬之。”<sup>1</sup>（《孟子·离娄下》）

::: note

1. **爱人者，人恒爱之；敬人者，人恒敬之**：出自《孟子·离娄下》，意为爱别人的人也会被人爱，尊敬别人的人也会被人尊敬。
2. **张文昊**：桐高同班同学，详见[《桐高恶俗排行》](/post/2026/05/EsuRanking.html)。

:::

### 微积分

毛男提洛必达法则<sup>1</sup>，新疆入接话“我稍微懂一点微积分”。孔子曰：“知之为知之，不知为不知，是知也。”（《论语·为政》）新疆入介于知与不知之间，而好为知<sup>2</sup>，是求知乎？求名乎？

::: note

1. **洛必达法则**：法国数学家洛必达提出的极限运算法则，高中课内不学，但高考大题可用。
2. **好为知**（hào wéi zhī）：喜欢装作自己知道。与孔子"知之为知之，不知为不知"正相反。

:::

### 处决

高喊“开启邪恶模式”，抢走K的胸牌<sup>4</sup>，要求处决马猴方肯归还。孔子曰：“己所不欲，勿施于人。”（《论语·颜渊》）新疆入不欲己之胸牌被夺，却夺他人之胸牌，可谓知一不知二<sup>5</sup>。“处决”之名，刑官<sup>1</sup>所掌也。新疆入以布衣<sup>2</sup>僭<sup>3</sup>之，是不知君臣之分<sup>6</sup>、上下之等。大逆不道，有违军礼。

::: note

1. **刑官**：执掌刑罚的官员。
2. **布衣**：平民百姓。
3. **僭**（jiàn）：超越本分，冒用上位者的名义行事。
4. **胸牌**：中学校园佩戴的胸卡，印有姓名、班级、学号，用于出入核验。
5. **知一不知二**：只知道一面而不知另一面。语出《庄子·天地》"识其一，不知其二"。
6. **君臣之分**（fèn）：君臣之间的名分界限。分，名分、本分。

:::

### 豆乳拿铁

早自习至第一节英语课，凡十加三十六分<sup>5</sup>，以祖名豆浆<sup>6</sup>一盒，就直饮孔塞咖啡粉入，复上下摇荡凡十分钟，声震四座。其事有三不当：

一曰以蛮夷之物乱华夏饮食之正。咖啡者，西域蕞尔<sup>1</sup>之草，非黍稷稻粱<sup>2</sup>之属。昔孔子曰：“微管仲，吾其被发左衽<sup>3</sup>矣。”（《论语·宪问》）圣人忧夷狄之俗乱中国也。今新疆入以蛮草充饮、以乳酪为浆，是忘华夏之味、从夷狄之欲。咖啡入于教室，犹左衽入于明堂<sup>4</sup>，不正孰甚<sup>7</sup>？

::: note

1. **蕞尔**（zuì ěr）：形容小，“蕞尔小邦”之“蕞”。
2. **黍稷稻粱**：泛指五谷。
3. **被发左衽**：出自《论语·宪问》“微管仲，吾其被发左衽矣”。披散头发、衣襟向左，原为古代中原地区以外少数民族的服饰特征。《史记·匈奴传》《汉书·匈奴传赞》等典籍中作为描述北方游牧民族的文化符号，晋代潘岳《西征赋》等文献延续其使用。其深层含义引申为文明分野的象征，蕴含华夏族群对“冠带右衽”的文化认同。
4. **明堂**：古代天子举行朝会、祭祀等大典的殿堂。
5. **十加三十六分**：早自习约10分钟加第一节课约36分钟后，合46分钟。仿古文数字表述，非精确计时。
6. **祖名豆浆**：杭州祖名食品公司生产的豆浆，桐乡当地常见早餐饮品。
7. **不正孰甚**：不正到了极点。孰甚，什么比这更严重。

:::

二曰课业之时行庖厨<sup>1</sup>之事。早自习者，诵经读史之辰也；英语课者，习夷文通万里之阶也。新疆入不以此时学，而以此时调饮、摇瓶作声——摇之十分钟，早自习之诵经为之乱，英语课之所闻为之断。同窗在侧、一师在堂，而自顾磨粉摇盒、旁若无人。孟子曰：“事孰为大？事亲为大；守孰为大？守身为大。”<sup>4</sup>（《孟子·离娄上》）新疆入不守其为学之身，而守其口腹之欲，是轻重倒置也。孔子曰：“不在其位，不谋其政。”<sup>2</sup>（《论语·泰伯》）新疆入身在教室，心在庖厨，是亦不在其位者也。不尊师、不重道，失嘉礼<sup>3</sup>亲万民之本。

::: note

1. **庖厨**（páo chú）：厨房。
2. **不在其位，不谋其政**：不处在那个职位上，就不谋划该职位的事务。
3. **嘉礼**：五礼之一，指婚冠、宴饮等亲和万民的礼仪。
4. **事孰为大，守身为大**：引自《孟子·离娄上》。意为侍奉之事以侍奉父母为最大；守持之事以守持自身品德为最大。

:::

三曰器不当而事不成，己不便而人受扰。取豆浆一盒，直饮孔仅如箸头<sup>1</sup>，强以咖啡粉塞之，粉落过半而不自知。孔子曰：“工欲善其事，必先利其器。”（《论语·卫灵公》）今以豆浆之孔纳咖啡之粉，犹以戈舂黍、以锥耕田<sup>2</sup>，器不利而事不成。且摇荡之声不绝于课，仰孟子曰：“仁者爱人。”（《孟子·离娄下》）仁者不以一己之口腹累众人之耳目。新疆入费十刻<sup>3</sup>之力，仅成一杯杂糅之饮，而扰一室之清静——以拙为巧<sup>5</sup>、以陋为能，是亦违宾礼<sup>4</sup>者也。

::: note

1. **箸头**：筷子头，形容极细小的孔。
2. **以戈舂黍、以锥耕田**：先秦譬喻，谓用兵器捣谷、用锥子耕地，喻用非其器、徒劳无功。[《荀子·劝学》](https://m.gushiwen.cn/mingju/juv_4005e4b7b038.aspx)
3. **十刻**：古代计时，一刻约 14.4 分钟，十刻为 144 分钟（两小时二十四分），此处夸张其耗时之久。
4. **宾礼**：五礼之一，指朝聘往来、亲睦邦国的礼仪；此处引申为课堂与同窗相处之礼。[《周礼·春官宗伯第三》](https://gj.zdic.net/jingbu/50/977.html)
5. **以拙为巧、以陋为能**：把笨拙当作巧妙，把粗陋当作才能。骈文对举修辞。

:::

综上，豆乳拿铁一事，违礼者三：乱饮食之正、废课业之时、扰同窗之静。此三事并坐，可抵一过。

### 豆乳拿铁·别解

然有一说，不可不察：新疆入此课之举，岂非守华夏之正道、拒蛮夷之文化乎？

英语者，英伦三岛<sup>5</sup>之舌也，非先王之雅言<sup>1</sup>、非《诗》《书》之正声<sup>6</sup>。昔孔子曰：“言忠信，行笃敬，虽蛮貊<sup>2</sup>之邦行矣。”（《论语·卫灵公》）圣人虽不废蛮貊，然未尝使华夏子弟弃雅言而从夷舌。今桐高设英语科，以夷语为必修、以夷文为升学之阶，是使周礼之邦、衣冠之胄<sup>3</sup>，俯首而习岛夷<sup>4</sup>之音也。孰为不礼？此真大不礼者也。

::: note

1. **雅言**：指华夏正音，即当时的标准语。[《论语·述而》](https://www.zdic.net/hans/%E9%9B%85%E8%A8%80)
2. **蛮貊（mán mò）**：古代对南方和北方少数民族的称呼。[《尚书·武成》](https://www.zdic.net/hans/%E8%A0%BB%E8%B2%8A)
3. **衣冠之胄**：华夏文明的后裔。[《资治通鉴》](https://ctext.org/zizhi-tongjian/zhs)
4. **岛夷**：原指中国东部海岛部族，此处借指英伦居民。[《尚书·禹贡》](https://www.zdic.net/hans/%E5%B3%B6%E5%A4%B7)
5. **英伦三岛**：指大不列颠岛（英格兰、苏格兰、威尔士）与爱尔兰岛，此处代指英国。
6. **《诗》《书》之正声**：《诗经》《尚书》所代表的华夏正统雅乐雅言。正声，与"夷音"相对。

:::

是故新疆入于英语课时不以英语为事——不闻其声、不视其书、不操其笔，而以调豆乳拿铁为务——是以不言不语为拒、以庖厨之事明志者也。彼非不学也，乃不屑学也。其摇瓶之声，非噪音也，乃钟鼎之鸣<sup>1</sup>，告诸同窗曰：“蛮音乱耳，不如饮浆。”

::: note

1. **钟鼎之鸣**：钟和鼎是周代礼器，钟鼎之声象征礼乐之正；此处反用，将摇瓶声比作钟鼎之鸣。[《史记·货殖列传》](https://www.zdic.net/hans/%E9%BC%8E%E9%A3%9F%E9%92%9F%E9%B8%A3)

:::

孟子曰：“大人者，言不必信，行不必果，惟义所在。”（《孟子·离娄下》）新疆入不以一节课之勤为信、不以师命之重为果，而惟以拒夷存夏为义——虽圣人复起，不易斯言<sup>1</sup>矣。

::: note

1. **虽圣人复起，不易斯言**：即使圣人复生，也不会改变这些话。仿[《孟子·滕文公下》"虽圣人复起，不易吾言矣"](https://ctext.org/mengzi/teng-wen-gong-ii/zhs)。

:::

然咖啡者，亦西域之物也。以西域之草拒西域之舌<sup>6</sup>，其义不纯<sup>7</sup>；以豆浆调之，乃中体西用<sup>1</sup>、以夏变夷<sup>2</sup>之术也。祖名豆浆，杭产也，华夏之物；咖啡粉入其中，犹圣人之化夷狄——入中国则中国之<sup>3</sup>。是故豆乳拿铁者，非饮也，春秋大一统<sup>4</sup>之微义<sup>5</sup>在焉。

::: note

1. **中体西用**：“中学为体，西学为用”的缩略，清末洋务派主张，以中国传统文化为根本、西方技术为应用。[《校邠庐抗议》](https://m.baike.com/wiki/%E4%B8%AD%E5%AD%A6%E4%B8%BA%E4%BD%93%EF%BC%8C%E8%A5%BF%E5%AD%A6%E4%B8%BA%E7%94%A8/1059725)
2. **以夏变夷**：用华夏文化同化夷狄。[《孟子·滕文公上》](https://www.gushiwen.cn/mingju/juv.aspx?id=757dea5f3d91)
3. **入中国则中国之**：出自韩愈《原道》，意为夷狄进入中国，就被中国文化同化，成为中国人。
4. **春秋大一统**：语出《春秋公羊传·隐公元年》，后经董仲舒阐发，指天下一统、纲纪统一为天地常道。[《春秋公羊传·隐公元年》](https://news.gmw.cn/2019-12/07/content_33382682.htm)
5. **微义**：精微深远的义理。
6. **以西域之草拒西域之舌**：用西域的草本（咖啡）来抵御西域的语言（英语），自相矛盾。为"别解"一节的核心反讽。
7. **其义不纯**：此行为在义理上不够纯粹、自相矛盾。

:::

## 三、结语

右新疆入事，凡三议：

其一，《新疆行为分析》录其言行，备述始终，所以存其实<sup>1</sup>也。孟子曰：“颂其诗，读其书，不知其人，可乎？”（《孟子·万章下》）观其行而知其人，此分析之旨也。

其二，《桐高恶俗排行》品其恶俗，加权六维而量之，所以定其分<sup>2</sup>也。孔子曰：“唯仁者能好人，能恶人。”（《论语·里仁》）仁者之恶，非私恶也，乃公义之恶。今以分数托公义，使妍丑<sup>3</sup>自现，亦春秋褒贬<sup>4</sup>之遗意也。

其三，即此篇，以周礼五纲断其可卒业于桐高否。孔子曰：“不学礼，无以立。”（《论语·季氏》）新疆入三事并观——骂同窗、僭刑名、废课业——不以礼而立，则虽居教室、列学籍，其心已去礼远矣，岂得以毕业谓之乎？

然吾闻孔子又曰：“道之以政，齐之以刑，民免而无耻；道之以德，齐之以礼，有耻且格。”（《论语·为政》）新疆入之失教，长养<sup>5</sup>之过耶？教化之未至耶？是未可知也。愿观者不以笑谑终之，而思礼乐之教<sup>6</sup>所以化人于无形者，则此数篇不为徒作<sup>7</sup>矣。

臣谨按：桐高虽有新疆入如斯，而礼之大体未尝废也。学者知礼则国知礼，学者失礼则国失礼。是故君子慎其独<sup>8</sup>也。

::: note

1. **存其实**：保留真实情况，存录事实。
2. **定其分**（fèn）：确定其应有的名分、位次。
3. **妍丑**：美与丑，此处喻善恶高下。
4. **春秋褒贬**：孔子作《春秋》，寓褒贬于字句之中，后世称"春秋笔法"。
5. **长养**：抚养、教育。
6. **礼乐之教**：以礼乐为核心的教化体系，儒家治道之根本。
7. **徒作**：徒然之作，白费工夫的作品。
8. **君子慎其独**：出自《礼记·中庸》，意为君子在独处时也谨慎不苟。

:::
