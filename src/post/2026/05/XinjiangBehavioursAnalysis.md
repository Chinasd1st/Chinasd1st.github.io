## 1. 欲盖弥彰；此地无银三百两

### 1.1 概述

新疆入常因为自己大脑算力不够等原因而做出欲盖弥彰的行为。我们从真实事例入手，深入浅出地分析他的这一行为特征：

#### 第五题我是抄我后桌的

数学学习到第八章立体几何，学校为巩固学生知识下发了一些综合卷；新疆入惊人地将第一面做得全对。结合他期中考试仅五十五分的数学成绩，不禁让人感慨其进步速度之快。讲评试卷时，新疆入突然说道：

:::center

**第五题我是抄我后桌的，其他都是我自己写的。**

:::

#### 微短剧

新疆入上个学期（2025年）在寝室说：“你们看微短剧吗？”

鸦雀无声。

**“反正我不看”。**

### 1.2 分析

在他说这句话（抄我后桌）前，没有人问过他怎么能做到全对，但他自己承认了抄后桌答案这一事实。这不禁让我们感受到其态度诚恳。

## 2. 恼羞成怒！不玩 = 反对

### 2.1 概述

熟悉新疆入的人都知道，新疆入对鸣潮这款游戏如痴如醉，几近到了疯魔的地步。本学期初（2026年3月），新疆入曾有过下列言论：

#### 发刀子

- **新疆入**：你知道鸣潮吗？
- **我**：我知道啊。
- **新疆入**：鸣潮不是在自媒体平台上有那种账号吗。
- **我**：你不用说的这么详细。
- **新疆入**：就是春节期间，鸣潮每天都连着发有关那个死了（类似表述）的角色的视频，游戏里面已经被刀了一遍了，现在还要来刀我，真是杀人诛心。

#### 玩不玩

> 本段在[新疆语录#玩不玩](/post/2026/04/XinjiangQuotation.html#%E7%8E%A9%E4%B8%8D%E7%8E%A9)一节中也有相关记述。

- **新疆入**：你玩不玩鸣潮？
- **张文昊**：不玩。
- **新疆入**：傻——逼。
  
顺我者昌，逆我者亡。

#### 水仙

- 新疆入：不知道怎么地我最近突然喜欢上了水仙；鸣潮不是有漂泊者吗，我最近就喜欢看男女漂泊者的视频。

#### 致敬

- **新疆入**：你肯定知道Rover这个词。
- **我**：我不知道。
- **新疆入**：你怎么可能不知道？
- **我**：这不是路虎的意思吗？
- **新疆入**：路虎那是音译！词典借我。

*新疆入在数学课查了十分钟词典还没有找到Rover一词，最后本人找到了这一词。*

- **新疆入**：找到了？
- **我**：对。
- **新疆入**：什么意思？
- **我**：流浪者。
- **新疆入**：差不多吧，漂泊者的意思。我在我个人其他游戏的昵称用的就是这个，算是一种致敬吧（原话如此）。

### 2.2 分析

新疆入曾提及他自己的手机使用受到限制，电脑也被没收，鸣潮似乎成了难得的慰藉！然而，新疆入这种立场先入为主的思想却十分耐人寻味！

## 3. 技术宅？

### 3.1 概述

新疆入是出了名的技术宅，在信息技术方面表现出了卓越的天赋，在新疆语录中我们也有提及。

#### 华硕

1. **新疆入是一个忠实的 ASUS 用户，其结晶纯度堪比某些管人粉丝。**

   他使用的游戏本是华硕天选5 i7 + 4050，疑似因为买不起 ROG 但是又放不下所谓的品牌信仰，只能购买其他产品线的产品。2024年他花费8000余元购买了这台笔记本，这一价格明显偏高：本人购买暗影精灵9 i9-14900HX + 4060无国补仅需8499元。同学想要购买一台笔记本，他首先便推荐购买 ROG 相关产品。听到对方拒绝购买 ROG 笔记本后，他说：其他厂商产品线我不熟，我只知道 ROG 的，我推荐你买魔霸（或类似表述）。

   不熟？**实则是怕知道其他品牌的高性价比后内心的信仰瞬间崩塌！**

2. **同学讨论电脑配置，新疆入十分鄙夷垃圾佬！**

   新疆入问对方电脑是什么主板，第一个候选项就是华硕；听到对方否认，直接说：

   **不要跟我说是很廉价的主板！**

   可见在他眼中，电脑品牌似乎只有 **“华硕”** 和 **“杂牌”** 两种品类，至于芯片组、PCH、供电、做工的客观分级则自然入不了他的法眼；让人不禁感叹品牌营销和新疆思维的双向奔赴竟能产生如此剧烈的化学反应。

   此外，同学还提及自己用的是七彩虹的4070，新疆入与同学一问一答：“华硕？”“不是。”“索泰？”“不是。”

   **话说索泰是二线品牌吗？**

   这句话是惊为天人的。黄仁勋研究了二十年 GPU 也没有想到，同一个新芯片、同一 PCIe、同一显存颗粒，分发到不同 OEM 手中性能竟会有如此差异。作为全球 AIC，论做工、散热、堆料，索泰比很多华硕丐版还强。而在新疆入眼里，只要不是华硕，连一线都算不上，甚至连二线都要打个问号！

#### Python

1. **新疆入自认为是Python大佬，实则是编程嘉豪**

   新疆入沉浸于初中时的地市级编程比赛一等奖的荣誉无法自拔，殊不知那一张试卷是学校自己批改的。新疆入上完信技课后对“jieba（结巴）”分词模块一概不知，还说“课本上没有相关用法”，可 jieba module 的相关函数及方法就在信技书数据处理的章末静静地躺着；不仅否认事实，他还强词夺理：

   **有个damn！**

   七宗罪第一罪：傲慢。如果不知道尚且作罢，别人提议却不听则是彻底的无知。说罢，他便想要在一体机上操作 PyCharm 进行“测试”。可找了半天他的“测试文件夹”也没有找到，我提议全局搜索，他却拒绝了。谁曾想，新疆入又爆了典：

   **我在家里都是用“IDEL”（应为IDLE）编程的，说实话对这种编译器（应为编辑器）不了解，我是最传统的。**

   他将 IDLE 念成 “IDEL”，又把“编辑器”与“编译器”混为一谈。不过这些细节似乎并未影响他的自信。“我是最传统的”——他说这句话时语气颇为郑重，仿佛自己并非不会使用 PyCharm，而是在坚守某种古典编程精神。相比之下，Vibe Coding 反倒显得没有那么后现代了。

   后来新疆入在我的协助下成功打开了一个PyCharm项目文件，我帮助输入了`import jieba`等语句，由于没有安装依赖，调试运行后shell会提示安装相关包，便点击安装，因为不确认是否安装成功，我便输入`pip install jieba`进行检查，而这台电脑上pip疑似不在环境变量中，导致显示没有相关指令，我便离开让新疆入自己操作。新疆入照葫芦画瓢输入了`pip ins...`便卡住了，原因是不会拼写 install 这一单词，不禁让人怀疑其在家日常编写的代码都是什么水平。

### 3.2 分析

在他那里，品牌似乎总带有某种道德意味。ROG、Python的象征含义早就已经超然于物外了。

## 4. 杂项

新疆入为游玩游戏边狱巴士，在2026年注册了谷歌账号，结果3月末刚注册没多久就被封号，提示多人同时注册（或类似表述）；这还不是最难绷的，他本人在三月与我有过如下对话：

- **新疆入**：你听过 Mili 这个“歌手”吗？

我佯装不知道。

- **新疆入**：ta 给边狱巴士写过歌。

众所周知 Mili 是一个音乐团体或乐队。后来我才意识到，在他的认知体系里 Mili 似乎是一个人。尽管单纯的咬文嚼字显得钻牛角尖，但是结合新疆入先前的行为模式，这一言辞引人发笑。同时他还有将两者强行绑定之嫌。

此外，他还说过：

:::center

**边狱巴士界面都是英语的，我一边拿翻译软件翻译一边玩；**

### 4.3 木柜子

英语必修二某一单元中有单词“band”。早读课上，新疆入看到后马上就坐不住了，连忙对我说：

:::center

**嘶.......你知道我现在一看到这个词就想到什么吗——少女乐队。**

:::

更幽默的是，英语课上，新疆入不知是意犹未尽还是海马体功能异常，将上面的话语一字未动地向我复述了一遍。尽管我两次都没有回复他，但是他似乎陷入自我中心主义的深渊难以自拔，亦或是正扮演《楚门的世界》中的主人公，将他思维最荒诞的一面恰如其分地绑在了耻辱柱上供世人鉴赏。恰如他强行向别人推荐ROG、科普鸣潮剧情一样，本质是自我中心主义的体现。

社会学家 Charles Derber 于其作品*The Pursuit of Attention: Power and Ego in Everyday Life*中提出**对话自恋**（Conversational Narcissism）一概念：通过“shift response”（转移回应）把话题焦点拉回自己，而非给予支持回应。这种行为常与脆弱自恋（Vulnerable/Covert Narcissism）重叠：表面寻求钦佩，内在高度敏感、易受伤，常采用受害者叙事来获取关注。[^13]

[^13]: Derber, Charles (1979). [*The Pursuit of Attention: Power and Ego in Everyday Life*](https://archive.org/details/pursuitofattenti00derb/mode/2up). New York: Oxford University Press.

### 4.4 上课回答问题

#### 2026.05.16 电场强度单位

新疆入曾扬言自己要在高三参加物竞。尽管他的物理原始分仅有三十几分，但这丝毫不影响他的好高骛远。物理课上讲述电场强度相关内容，老师使用了 $\mathrm{V/m}$ 而非 $\mathrm{N/C}$ 作为场强单位。老师随口一提你们知道是怎么推导出来的吗，新疆入迫不及待地说：“电势除以米”，然而相关推导需要等到学习电势差后才能得出，这一表述极不严谨。这一事例恰恰印证了新疆入发言的几大特征：一是望文生义，糊弄玄虚，不懂装懂，答案的正确与否并不重要，重要的是能否争夺到话语权；二是故作高深，虚张声势，急于表现自己，却暴露了自己的无知与浅薄。至于所谓“高三搞物竞”，只要他说出了这句话，他就觉得自己已经和那些普通的差生不一样了，自己是一个有追求的人。

### 4.5 学习热情

新疆入学期初曾有过短暂的“努力学习”的时光，尽管这些努力的成效转换率似乎无限趋近于零，但是他似乎认为努力和成功中间有一个显而易见的等号。

后来新疆入似乎也发现了成效转化率过低的现象，他的学习成绩不仅没有提高，他本人也感到身心俱疲。他便说：

:::center

**我发现我前些天学习热情过度高涨了，导致我现在非常疲累。**

:::

### 4.7 新疆饮料

本文不讨论过量摄入赤藓糖醇等代糖的副作用，仅针对新疆饮料这一现象展开叙述。

### 4.8 游戏开发；华南理工

**7班地理老师何娅**：班里的男生不要幻想搞什么游戏开发。

### 4.9 足球

可见新疆入还是有成为迪哥的潜质的。

## 6. 拓展阅读

### 6.1 自恋型人格相关研究[^3]

:::warning

原文应为“自恋型人格障碍的表现”；“障碍”是严肃心理学用语，在此处使用显得不恰当，故略去。

:::

1. **给人以傲慢自大的印象**：自恋型人格障碍个体无理性地认为自己拥有特殊天赋和特权，藐视社会规范和他人权利，表现出自负、自夸和傲慢；
2. **以利用、掠夺为目的地与他人交往**：自恋型人格障碍个体利用他人以达成个人目的，没有同情心，不考虑他人权利和感受，导致人际关系疏远；
3. **以自我中心、散漫随意为特征的思维方式**：在不利情形下，自恋型人格障碍个体忽视逻辑，随意想象，夸大自身能力，将失败幻想为成功，并通过文饰作用维持自尊；
4. **以满不在乎、自我感觉良好为特点的情绪体验**：自恋型人格障碍个体通常自我感觉良好，即使遭遇打击也能迅速通过文饰作用来恢复自信；
5. **夸大的自我评价**：自恋型人格障碍个体觉得自己是令人欣佩的、最优秀的，深信自己是卓越和值得特殊待遇的，对他人的怀疑表示蔑视和否定，认为自己可以超越所在社会或文化的准则、习惯、习俗和伦理道德；
6. **以文饰为主的防御机制**：面对挫折或不符合社会规范的行为，他们无意识地用有利于自己的理由歪曲事实，来减轻焦虑，从而保护了非理性的自我形象和自信心。

### 6.2 邓宁-克鲁格效应

邓宁-克鲁格效应（英语：Dunning-Kruger effect），简称邓克效应或达克效应（DK effect），亦有人称井蛙现象，是一种认知偏差，能力欠缺的人有一种虚幻的自我优越感，错误地认为自己比真实情况更加优秀。美国康乃尔大学的社会心理学家大卫·邓宁和贾斯汀·克鲁格将其归咎于元认知上的缺陷，能力欠缺的人无法认识到自身的无能，不能准确评估自身的能力。他们的研究还表明，反之，非常能干的人会低估自己的能力，错误地假定他们自己能够很容易完成的任务，别人也能够很容易地完成。[^4]但能力欠佳的人对自己能力的评估并不比能力较佳的人高。[^5][^6]

### 6.3 确认偏误

确认偏误（英语：confirmation bias，也称 confirmatory bias、myside bias[^7] 或 congeniality bias）[^8]，是指人们倾向于以能够证实或支持自己既有信念、价值观[^9] 或决定[^10] 的方式去搜索、解释、偏好以及回忆信息。人们会主动选择支持自身观点的信息，而忽视相反的信息；或者在面对模糊证据时，将其解释为支持自己原有立场的依据。

这种效应在涉及期望结果、情绪化议题以及根深蒂固信念时最为明显。确认偏误往往来源于自动化的心理习惯。研究一再发现，人们倾向于以单方面的方式检验观点，主要寻找能够支持既有假设的证据。关于“选择性接触”（selective exposure）的研究也表明，人们在维护自身态度方面存在差异，其中一些人会更强烈地抵制与自身观点相冲突的信息。[^11][^12]
