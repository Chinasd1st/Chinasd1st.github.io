# 化学恐怖指数：当王水、氰化物与硝酸甘油走进教室

:::info

ChatGPT死活不让我生成海报，只好自己花五分钟手搓了一个！，，

:::

化学课是高中最危险的课。不是因为实验室里有浓硫酸，而是因为教室里有1977、小Liar和新疆入。

当老师在讲台上讲氮氧化物的时候，1977已经在脑子里完成了从NO到王水的犯罪方案迭代；当老师讲到氰化物的毒性的时候，小Liar正在怂恿吴嘉成去问一个不该问的问题；当朱正伟讲到硝酸甘油既是药物又是炸药的时候，新疆入已经开始向后桌传授他的"化学军工"知识了。

这三个人有一个共同点：他们都把化学课当成了犯罪预备课，区别只在于——1977是编剧，小Liar是导演，新疆入是那个自带BGM的群众演员。

本文基于[《桐高幽默事件》](/post/2026/01/HumorousIncidents.html)（2026/01）、[《桐高恶俗排行》](/post/2026/05/EsuRanking.html)（2026/05）及相关课堂记录，对三起化学相关恶俗事件进行拆解与评分。评分沿用六维体系（虚荣/装腔、无知自信、侵犯边界、自我中心、言行低俗、恶意攻击），并新增两个化学专属维度：**化学知识准确度**与**犯罪想象力**。

## 一、1977：王水毒害姝娴

> 又名王水男、跳水男、水男

1977的化学恐怖主义不是突然出现的。它有一个清晰的升级链，每一步都比上一步更离谱，直到你意识到——这个人把化学课上学到的东西，全部转化成了一套犯罪方案。

### 1.1 犯罪方案迭代

事情的起点是化学课讲了NO和NO₂。对大多数同学来说，这是一节关于氮氧化物性质的普通课。但对1977来说，这是一次灵感的降临。

下课后，1977开始向众人讲述自己曾进行稀硝酸实验的"辉煌事迹"。K见状，为了套出他的美妙话语供自己轻哼用，故意设诱道：

:::center

**"你往王珠娴办公室通点NO。"**

:::

这句话是一枚钩子。钩子一扔，鱼就咬上来了。

1977先是一愣，随即用一种顿悟而激动的语气高声喊道：

:::center

**"A，对哦！"**

:::

此刻，他仿佛化身为一战中德国的毒气兵，试图奔赴前线——也就是办公室——去"往那里通NO和NO₂"。旋即，他的大脑像是被接通了电路，一个灯泡在脑海中亮起：历史上那些风流而危险的炼金术士与化学家的英姿在他眼前浮现，目光中闪烁着近乎虔诚的希冀。

但他没有止步于NO。他的大脑开始自动升级方案：

| 迭代 | 方案 | 1977的评价 |
| :---: | ---- | ---------- |
| v1 | 通NO | "对哦！" |
| v2 | 通NO₂ | "NO₂有颜色，否则一看就看得出来" |
| v3 | 抹浓硫酸 | "薄薄的一层，这样她被烫伤了都看不出来" |
| v4 | 用王水 | "酸性不够强，体现不出我对王珠娴的怨恨" |

从一氧化氮到王水，四个版本，每一步都在上一步的基础上优化了隐蔽性和伤害性。这不是随口说说——这是一次完整的犯罪方案迭代。v1考虑的是"能不能做"，v2考虑的是"会不会被发现"，v3考虑的是"怎样不留痕迹"，v4考虑的是"怎样让效果最大化"。

:::warning

v2和v3之间有一个值得注意的细节：1977说"薄薄的一层"浓硫酸。这说明他在想象这个场景的时候，连涂抹的手法都想好了——不是泼上去，是薄薄地抹。这个细节暴露出他思考的深度：他不是在开玩笑，他是在认真规划。

:::

### 1.2 王水的化学真相

1977最终选择的武器是王水。他说"应该用王水，不然酸性不够强"——这句话看似在炫耀化学知识，实际上暴露了他对王水的误解。

王水（Aqua Regia），由浓硝酸和浓盐酸按1:3（物质的量比）混合而成[^2]：

$$\mathrm{3HCl + HNO_3 = NOCl + Cl_2 + 2H_2O}$$

王水的核心能力不是"酸性强"——它的酸性甚至不如浓硫酸。王水的真正威力在于**氧化性**：它能溶解金和铂这类惰性金属，这是浓硫酸做不到的。但1977把王水选为"怨恨的载体"，理由是"酸性不够强"——这就像选了一把狙击步枪来切菜，因为它"比较长"。

这不是知识的运用，这是知识的误用。1977把王水从冶金工业的工具降格为想象中的犯罪武器，同时连它为什么厉害都说错了。

### 1.3 "魔酸的酸性比硫酸强十几亿倍"

炫耀完王水之后，1977意犹未尽，转向K、李奕阳等人，开始系统性地炫耀自己"掌握"的化学知识。他一口气报出了十几种酸的名字，强弱交错、顺序全凭兴致。每提到一种无机酸，他的情绪便随之抬升几分，越说越来劲，神情中满是自信与满足，仿佛这些化学式本就属于他的个人履历。

谈到最后，他更是压低声音、郑重其事地抛出一句总结：

:::center

**"魔酸的酸性比硫酸强十几亿倍。"**

:::

这句话一出，既像是在给众人科普，又像是在为自己的学识盖章认证——至于"十几亿倍"从何而来、是否真有比较依据，已不在他的考虑范围之内；重要的是，在那一刻，他说得极其确信。

实际上，魔酸（Magic Acid，HSO₃F·SbF₅）的哈米特酸度函数 H₀ ≈ −23，而浓硫酸的 H₀ ≈ −12。按H₀标度计算，魔酸的酸性确实比浓硫酸强约10¹¹倍——也就是一千亿倍，不是"十几亿倍"。1977的数量级差了两个数量级，但这不影响他说话时的自信程度。

:::info 附：化学知识准确度对比

| 1977的说法 | 实际情况 | 准确度 |
| ---------- | -------- | :----: |
| 王水酸性强 | 王水的氧化性强，酸性不如浓硫酸 | ❌ |
| 魔酸比硫酸强十几亿倍 | 强约一千亿倍（10¹¹倍） | ⚠️ 量级偏差 |
| 报出十几种酸名 | 顺序全凭兴致，无比较基准 | ❌ |

:::

### 1.4 恶俗维度分析

1977在化学恶俗事件中的表现，可以用六维体系做如下拆解：

- **虚荣/装腔（15分）**：报酸名、讲魔酸，本质是在用化学术语构建"我很懂"的人设。但他选错了听众——在场的人没有一个需要他科普。
- **无知自信（20分）**：王水的选型理由错误，魔酸倍数偏差两个数量级。这不是"我知道但说错了"，这是"我真的以为自己是对的"。
- **侵犯边界（25分）**：把对老师的犯罪幻想在教室里公开说出来，且对象是具体的人（王珠娴）。
- **自我中心（15分）**：整个过程是1977的独角戏——从"辉煌事迹"到"化学科普"，他始终在享受被关注的感觉。
- **言行低俗（10分）**：用犯罪幻想当谈资，品味问题的硬指标。
- **恶意攻击（20分）**：虽然没有实际行动，但"薄薄一层浓硫酸"的规划已经超出了玩笑的边界。

**加权综合：19.7**

## 二、小Liar：氰化物试探

> 又名钱东成，分班前同学，现已不在当前故事现场

如果说1977是化学恐怖主义的编剧——他自己写剧本、自己念台词、自己享受表演——那小Liar就是化学恐怖主义的导演：他自己不出镜，但他知道怎么让别人替他演。

### 2.1 怂恿提问

2025学年第一学期，化学老师吴诚带领同学们走进实验室进行检验三价铁离子（Fe³⁺）的实验，使用的试剂包含硫氰化钾（KSCN）。就在这个当口，小Liar让他的同学吴嘉成前去询问吴诚：

:::center

**"氰化物是不是有剧毒？"**

:::

吴诚的反应出乎所有人意料。他没有回答，而是直接反问了一句：

:::center

**"你想干什么？"**

:::

气氛一度十分尴尬。

这不是小Liar自己想问的。他比谁都清楚这个问题问出来的后果——在检验Fe³⁺的实验课上突然问氰化物的毒性，就像在手术室里问"手术刀能不能杀人"。技术上没错，但时机和场合全错了。他选择让吴嘉成去问，而不是自己问——这说明他知道这个问题的分量。

这是一次精心设计的**信息投毒**。小Liar不需要自己动手，他只需要在吴嘉成耳边说一句"你去问问老师氰化物是不是剧毒"，就能达成三个效果：

1. **制造话题**：让"氰化物"这个词在实验室里被说出来
2. **转移风险**：问题不是他问的，老师追查也追不到他头上
3. **满足好奇**：他确实想知道答案，但他不想自己承担提问的代价

:::center

**这比1977的王水计划更隐蔽——因为1977至少是自己在说，而小Liar是让别人替他说。**

:::

### 2.2 吴诚的反问

氰化物（CN⁻）确实是高中化学的教学内容之一。它涉及的知识点包括络合物、配位键、以及工业应用（如电镀、冶金）。但在检验Fe³⁺的实验课上突然问氰化物的毒性，时机和场合都错得离谱——就像在消防演习的时候问"怎么放火"。

吴诚的反问"你想干什么？"是一次漂亮的课堂危机处理。他没有回答问题，也没有回避问题，而是用一个问题把提问者逼到了墙角：

- 如果吴嘉成说"我就是好奇"，吴诚可以追问"在做Fe³⁺实验的时候好奇氰化物？"
- 如果吴嘉成说"有人让我问的"，那就直接暴露了小Liar
- 如果吴嘉成不回答，问题自动消解

小Liar的计划在这一步就已经失败了。他低估了教师的反应速度——不是所有老师都会老老实实回答学生的问题。

### 2.3 恶俗维度分析

- **虚荣/装腔（3分）**：没有明显的虚荣表现，他的动机是实用主义的。
- **无知自信（5分）**：他对氰化物有一定了解（否则不会想到这个问题），但对提问场合的判断严重失误——在Fe³⁺实验课上问氰化物毒性，暴露了他对课堂情境的无知。
- **侵犯边界（15分）**：怂恿吴嘉成提问，把风险转嫁给别人，这是典型的"借刀杀人"式越界。
- **自我中心（12分）**：他的好奇心优先级高于课堂秩序和同学的利益。
- **言行低俗（5分）**：虽然没有直接的低俗言论，但"怂恿提问"本身是一种低品质的社交行为。
- **恶意攻击（8分）**：没有明显的攻击意图，但让吴嘉成在全班面前被吴诚反问"你想干什么？"，客观上造成了一次社交伤害。

**加权综合：8.8**

:::warning 备注

小Liar的化学恶俗指数不高，是因为他的行为模式与1977和新疆入不同。1977和新疆入是"主动输出"型——他们自己说、自己表演、自己暴露。小Liar是"被动触发"型——他不出面，不署名，不留证据。这种隐蔽性让他的恶俗指数看起来低，但实际影响并不小。吴嘉成在全班面前被吴诚反问"你想干什么？"的尴尬，就是小Liar隐身术的代价转嫁。

:::

## 三、新疆入：硝酸甘油与张雪机车

> 又名达克效应输出永动机，化学课上的定时炸弹

新疆入的化学恐怖主义与1977不同。1977是犯罪编剧，小Liar是信息投毒者，而新疆入是——**课堂广播站**。他的武器不是具体的犯罪方案，而是把课堂上听到的每一个关键词都变成他的表演素材。

### 3.1 氮氧化物课上的连环爆典

2026年6月15日下午第一节化学课，化学老师朱正伟讲解氮氧化物。朱正伟提到N₂O是笑气："笑气肯定有毒的咯，稍微吸几口人会哈哈大笑，吸多了就会中毒。"又提到一氧化氮在"伟哥（西地那非）"中的应用："吃下去之后会释放NO来扩张血管。"[^1]

[^1]: 朱正伟此处的表述有简化。西地那非（Sildenafil）是高选择性的PDE5抑制剂，它能暂时"麻痹"这个分解酶，让cGMP的水平升高并维持更久，从而放大和延长一氧化氮的血管舒张信号，而非直接"释放NO"。

[^2]: 王水的配比常见两种表述：物质的量比 HNO₃:HCl = 1:3（维基百科），或体积比 HCl:HNO₃ = 3:1（百度百科、科普网）。两种表述等价，但混用易引起混淆。本文采用物质的量比。

[^3]: 此处引用为新疆入学语原话。张雪确为凯越机车联合创始人（2017年创立，2024年离开后创办张雪机车），但"外国人评价'调教跟吃了三片伟哥一样硬'"这一具体表述无公开出处，可能为新疆入的即兴加工或道听途说。

这些本来是氮氧化物的常规教学内容。但新疆入听到之后，立刻进入了表演模式。

#### 张雪机车

朱正伟讲到NO的应用时，新疆入突然插话：

:::center

**"张雪不是原来是凯越机车的吗？他把这个机车运到香港那边，外国人试了直接评价'这辆车的调教跟吃了三片伟哥一样硬'。"[^3]**

:::

这句话的恶俗之处不在于内容本身——用"伟哥"比喻发动机调教虽然粗俗，但勉强算是一种民间修辞。问题在于它出现的场合：这是在化学课上，老师正在讲解一氧化氮的生理作用，新疆入把"伟哥"和"机车"缝合在一起，制造了一场课堂上的尴尬。

#### 硝酸甘油

朱正伟继续讲解氮氧化物，新疆入接连说出：

:::center

**"硝酸甘油。"**

**"制备硝酸甘油。"**

:::

这不是在回答问题，不是在讨论化学性质，而是在**公开传播**——确保他周围的人都知道"硝酸甘油就是炸药"。"制备"二字更是直接从"知道"升级到了"如何制造"，动机不言自明。

### 3.2 达克效应的化学变体

新疆入在化学领域的表现，与他在其他领域的表现如出一辙——**用有限的知识构建无限的自信**。

回顾新疆入的语录链：

| 语录 | 来源 | 本质 |
| ---- | ---- | ---- |
| "不考物化我就起飞了" | 期中考前 | 逃避 |
| "我稍微懂一点微积分" | 毛男提到洛必达法则后 | 达克效应 |
| "IDE最传统" | 编程课 | 认知盲区 |
| "张雪机车跟吃了三片伟哥一样硬" | 化学课 | 联想过载 |
| "硝酸甘油""制备硝酸甘油" | 化学课 | 知识碎片的放大 |

从数学到编程再到化学，新疆入的模式始终一致：听到一个关键词→调用有限的先验知识→以自信的语气向周围人输出→对输出内容的准确性和场合的恰当性不做任何判断。

在化学课上，他的"先验知识"可能仅限于"硝酸甘油可以做炸药"这个事实。但他不满足于知道这个事实——他要让所有人都知道他知道这个事实，而且是在最不合适的场合说出来。

### 3.3 课堂干扰的累积效应

新疆入在化学课上的表现，单独看是一次普通的学生走神。但放在整个学期的背景下，这是一系列持续性课堂干扰的一部分。

他在数学课上说过"不考物化就起飞了"，在编程课上混淆编译器和编辑器，在化学课上把NO的应用话题引向伟哥和机车，然后公开提及硝酸甘油的制备。每一次干扰都不是致命的——老师不会因为他提了一嘴硝酸甘油就停课。但当这些干扰累积起来，它们构成了一种**噪声环境**：周围的同学需要不断过滤他的输出，老师需要不断把注意力拉回教学内容。

这就是新疆入式恶俗的核心——**不是一次性的越界，而是持续性的消耗**。

### 3.4 恶俗维度分析

- **虚荣/装腔（12分）**：在同学面前展示"我知道硝酸甘油是炸药""我懂伟哥的原理"，满足"我什么都懂"的虚荣。
- **无知自信（18分）**：他的化学知识仅限于碎片化的事实，但他用这些碎片构建了一个"化学达人"的人设。
- **侵犯边界（15分）**：在课堂上公开提及炸药制备和伟哥，扰乱教学秩序，侵犯了师生共同维护的课堂空间。
- **自我中心（15分）**：他的表达欲优先级高于课堂纪律。
- **言行低俗（12分）**：把"伟哥"和"机车"缝合在一起，用"制备硝酸甘油"当谈资，品味问题的硬指标。
- **恶意攻击（5分）**：没有明显的恶意，但破坏教学秩序的效果是客观存在的。

**加权综合：12.8**

## 五、化学教师视角

王珠娴是数学老师。她从头到尾不知道1977对着K和作者摊开的那个化学武器计划书里，自己的名字成了核心关键词。

吴诚是化学老师。他在检验Fe³⁺的实验课上被小Liar怂恿的同学问了一个关于氰化物毒性的问题，他用一句"你想干什么？"化解了危机。

朱正伟也是化学老师。他在讲解氮氧化物的课堂上被新疆入的"张雪机车"和"制备硝酸甘油"打断了教学节奏。

从教师的角度看，这三起事件构成了一幅完整的"化学课堂失控图"：

- **1977**代表的是**犯罪幻想型**：学生把课堂知识转化成了伤害他人的想象，且不加掩饰地公开输出。教师需要处理的不仅是知识纠错，还有安全隐患。
- **小Liar**代表的是**信息投毒型**：学生利用课堂提问机制，把敏感话题引入教学空间。教师面临的困境是——回答会偏离教学计划，不回答会显得回避问题。
- **新疆入**代表的是**噪声干扰型**：学生不断在课堂上插入无关信息，消耗教师的注意力和课堂时间。单次干扰不致命，累积起来足以影响教学节奏。

三者共同指向一个问题：**化学课的"危险性"不在于化学试剂，而在于使用化学知识的人。**

## 结语

1977的王水方案最终停留在了口头。小Liar的氰化物提问被吴诚一句"你想干什么？"挡了回去。新疆入的硝酸甘油和张雪机车最终也只是两段课堂插曲。

但这些事件的意义不在于后果的严重程度，而在于它们揭示的行为模式：

- 1977展示了**犯罪想象力**——一个人可以把课堂知识转化为伤害他人的方案，且在转化过程中毫无心理负担。
- 小Liar展示了**隐蔽性**——一个人可以在不暴露自己的前提下，让他人替自己完成一次信息投毒，而当老师反问"你想干什么？"的时候，承受尴尬的是被利用的吴嘉成，不是他自己。
- 新疆入展示了**持续性**——一个人可以用碎片化的知识不断干扰教学秩序，且浑然不觉。

化学知识本身无罪。王水可以用来溶解金属做冶金分析，氰化物可以用来电镀和提炼黄金，硝酸甘油可以用来治疗心绞痛。但当这些知识被用来"规划"伤害、试探边界、或制造恐慌时，知识就变成了武器——不是化学的武器，而是社会学意义上的武器。

这不是一篇关于化学的科普。这是一份关于"用化学作恶"的行为样本。
